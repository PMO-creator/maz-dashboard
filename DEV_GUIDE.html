Se o dashboard abrir em branco após uma mudança
Não entre em pânico. Peça ao Claude para investigar. As causas mais comuns são:
O código ficou incompleto ao ser editado
Diga ao Claude: "O dashboard abriu em branco, pode verificar se o arquivo de código está completo e sem erros?"
Erro de digitação no código
Diga ao Claude: "O dashboard ficou em branco após a última alteração, pode revisar o que foi mudado?"

Como o dashboard calcula o status de cada nível
💡
O status de um Marco, Grupo ou Eixo é calculado automaticamente com base nas suas tarefas — sempre usando o pior status encontrado.
📊 Eixo herda o pior status dos seus Grupos
📁 Grupo herda o pior status dos seus Marcos
🎯 Marco herda o pior status das suas Tarefas
Exemplo: se um grupo tem 10 tarefas "Feitas" e 1 "Atrasada" → o grupo aparece como Atrasado.
